# 0.1.0b1 — first public beta

Initial public beta of OpenACA. Backfilled retroactively to populate the GitHub Releases page; the package was shipped to PyPI on 2026-05-18 from this tag.

## Highlights

- **CLI ergonomics.** `openaca --version` and short_help blurbs added to every subcommand so users can self-orient.
- **Beta program rails.** Beta-tester guide, DM-only reporting path while the repo is private, beta-feedback issue template (blank issues disabled), and PyPI-safe README that won't break the project page during the closed beta.
- **Docs polish.** Beta status surfaced in README, prerequisites and a 30-second try-it-out path, links into the `openaca-demo` repo for additional fixtures.

## Install

`uv tool install openaca==0.1.0b1` or `pip install openaca==0.1.0b1`.

## Compatibility

Pre-V0 beta — no back-compat hedging. CLI surface and JSON output may change between betas.
